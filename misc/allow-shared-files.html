Allow Shared Edits by Default

By default, files you create cannot be edited by others. To permanently allow others to edit your files in the idea-repository folder, do the following:

  • Open a terminal.
  • Run the command: export EDITOR=nano && crontab -e
  • Paste the following on 2 lines (you can't use ctrl+v):
    * * * * * chmod 775 -R /var/www/html/idea-repository/*
    * * * * * chgrp -R idea /var/www/html/idea-repository/*
  • ctrl+o to save.
  • Enter to confirm.
  • ctrl+x to exit.

Verify

Run crontab -l and it should output * * * * * chmod 775 -R /var/www/html/idea-repository/* && chgrp -R idea /var/www/html/idea-repository/*
