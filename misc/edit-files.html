Editing Files on the Web Server

The Idea Repository code lives at /var/www/html/idea-repository on the CIT web server.

Linux

If you are using Linux, your file manager should be able to mount the files onto a local folder. Google "connect to SFTP share [Linux distribution name]" and you should be able to find it. From there, you can open it just like a normal folder in VSCode.

Windows

  1. Download WinSCP
  2. Once it is installed, open it enter the server connection details:
    • Protocol: SFTP
    • Host: cit.wvncc.edu
    • Port: 2080
  3. After selecting "Login" and entering your password, navigate to /var/www/html/idea-repository.
  4. Now you will need a folder to download the code into. Create one at \Desktop\idea-repository or similar.
  5. Back in WinSCP, use Ctrl+S to save into the folder you just created.
  6. Select "Local" as the target directory and click "OK" to both dialog boxes.
  7. Open the target directory in VSCode and make sure it is up to date with the server.
  8. In WinSCP, Press Ctrl+U to sync your folder with the server.
  9. Click "Start" to link your local folder with the server.
  10. Back in VSCode, changes made to your folder will transfer to the server automatically.

Multiple Users

Multiple users cannot edit the same sime simultaneously. You may override each other's edits! We may need to use Git or have designated edit times.
